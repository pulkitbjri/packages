# chat-sdk

Shared React Native chat UI and Firestore helpers for consumer and partner apps.

## Local package

This package is vendored locally under `packages/chat-sdk`.

Use it from an app with:

```json
"chat-sdk": "file:../packages/chat-sdk"
```

Then run `npm install` in the app.

## Peer dependencies

See `package.json` for the required `react`, `react-native`, and `@react-native-firebase/*` peers.
